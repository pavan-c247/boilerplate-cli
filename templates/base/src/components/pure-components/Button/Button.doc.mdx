## Use cases

### Primary

Use the primary button for an action that the user is most likely to take. In most situations this button should only be used once per page.

### Secondary

Use a secondary button when there is more than one action on the page. There can be more than one secondary button.

### Tertiary

Use tertiray button when its required to assign less weightage to an action. E.g. In a filter options popup, the primary action will be “Apply”, the secondary action will be “Cancel” and “Clear”, or “Reset” will be the tertiary action.

### Icon

Use the primary button for an action that the user is most likely to take. In most situations this button should only be used once per page.

### Icon with border

Icon buttons should be used for actions that have lower weightage, or in situations where there is not enough place to show button with text. Another reason for usage could be asthetics.

### Danger

Use the primary button for an action that the user is most likely to take. In most situations this button should only be used once per page.
